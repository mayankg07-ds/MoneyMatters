# Indian Stock Symbols - Yahoo Finance Format

## NSE (National Stock Exchange)
Format: `SYMBOL.NS`

| Company | Symbol | Yahoo Format |
|---------|--------|--------------|
| Reliance Industries | RELIANCE | RELIANCE.NS |
| TCS | TCS | TCS.NS |
| Infosys | INFY | INFY.NS |
| HDFC Bank | HDFCBANK | HDFCBANK.NS |
| ICICI Bank | ICICIBANK | ICICIBANK.NS |
| Wipro | WIPRO | WIPRO.NS |
| ITC | ITC | ITC.NS |
| State Bank of India | SBIN | SBIN.NS |
| Bharti Airtel | BHARTIARTL | BHARTIARTL.NS |
| Asian Paints | ASIANPAINT | ASIANPAINT.NS |

## BSE (Bombay Stock Exchange)
Format: `CODE.BO`

| Company | BSE Code | Yahoo Format |
|---------|----------|--------------|
| Reliance Industries | 500325 | 500325.BO |
| TCS | 532540 | 532540.BO |
| Infosys | 500209 | 500209.BO |
| HDFC Bank | 500180 | 500180.BO |
| ICICI Bank | 532174 | 532174.BO |

## How to Find Symbol
1. Go to Yahoo Finance: https://finance.yahoo.com
2. Search for company name
3. Symbol shown in format: `RELIANCE.NS` or `500325.BO`
